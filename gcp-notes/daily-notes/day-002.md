# Before everything understand where everything lives.
|Directory | what lives here |
|\ | root- top of everything |
|/home | user home directories |
|/etc | Configure files |
|/var | variable data-logs,databases |
|/tmp | temporarary files, cleared on reboot |
|/usr | user program & utilities |
|/bin | essential command binaries |
|/sbin | system binaries(root only) |
|/proc | virtual file system-running processes|
|/dev | device files |
|/opt | optional-third party software |
|/root | home directory of root user |

----

## / - root directory,starting point for entire file system.Every single file & directory on this system is located under it.
